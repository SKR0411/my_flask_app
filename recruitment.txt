#About the Project
-----------------
This project is a Python-based Expense Tracker designed to help users record,
manage, and analyze their daily expenses. It provides simple functionality
for adding expenses, storing them in a file/database, and retrieving reports.

#Requirements
------------
- Python 3.8 or above
- No external libraries required (only built-in modules)

#Installation & Usage
--------------------
1. Clone the repository:
   git clone https://github.com/SKR0411/Python.git

2. Navigate to the project directory:
   cd Python

3. Run the script:
   python expense_exe.py

#Features
--------
- Add new expenses with amount, category, and date.
- View all past expenses in a structured format.
- Generate basic summaries (e.g., total spent, category-wise distribution).
- Easy to use with a command-line interface.
